Veille Technologique

Electron

Electron est un framework développé par Github (Microsoft) permettant de développer des applications multi-plateformes de bureau avec des technologies web

Site officiel

Présentation

Qu'est ce qu'un framework ?

L’objectif d’un framework est de simplifier le travail des développeurs informatiques, en leur offrant une architecture “prête à l’emploi” et qui leur permet de ne pas repartir de zéro à chaque nouveau projet

Fonctionnement d'Electron
Les avantages d'Electron
  • Création d’applications “portables” : sans aucune installation sur le système client. Donc on n’installe pas d’application tierce comme Java qui s’exécute en permanence en tâche de fond pour vous demander d’installer la dernière mise à jour.
  • Un support et/ou de l'aide en cas de besoin sur leur discord
  • On peut intérger notre application sur le Mac Store et/ou sur le Windows Store
  • Mise a jour régulière
Les inconvénients d'Electron
  • Il prend beaucoup d'espace de stockage même pour un petit projet (entre 50 et 150 Mo)
  • Il utilise un navigateur web embarqué (Chromium) pour afficher un simple message, il consomme beaucoup de ressources (RAM et CPU)
Logiciel utilisant Electron

  • L'application Microsoft Teams
  • L'application Skype
  • L'application Twitch
  • L'application WhatsApp
  • Visual Studio Code
  • L'application Messenger
